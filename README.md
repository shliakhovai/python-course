## Огляд завдань

### 1. Змінні та базові обчислення
Конвертер валют і конвертер часу. Знайомство з константами (`UPPER_CASE`), операцією `input()`, перетворенням типів та форматуванням виводу.

### 2. Рядки та колекції
П'ять задач: логування користувача (ініціали), маскування email, симуляція `set()`, аналіз тегів через `union`/`intersection`, обробка рядка з числами.

### 3. Цикли, словники та Git
Інтерактивний калькулятор з циклом та облік витрат за категоріями. Також розглядається правильна послідовність роботи з гілками (`git checkout`, `git stash`).

### 4. Функції та валідація даних
Консольна програма збору даних користувача: валідація імені, пароля, email з `raise ValueError`, маскування та шифрування за прапорами.

### 5. CRUD та структури даних
Система постів з авторизацією: заборонені слова, видалення та редагування власних постів, перегляд стрічки, підписки, лічильник `views`, коментарі.

### 6. Об'єктно-орієнтоване програмування
Система бронювання місць у кінотеатрі: клас `CinemaHall` з інкапсуляцією та `@property`. Додаткове завдання — `VIPCinemaHall` з наслідуванням і `super()`.

### 7. Асинхронне програмування
Симулятор рейду: кілька гравців асинхронно атакують боса через `asyncio.gather()`. Захист спільного стану замком `asyncio.Lock()` для уникнення Race Condition.

### 8. Telegram Bot
Посилання на репозиторій з реалізацією: [atom-space](https://github.com/shliakhovai/atom-space).

### 9. Pre-commit hooks
Налаштування pre-commit hooks для автоматичної перевірки коду перед кожним комітом.

## Загальні вимоги до здачі

- Кожне завдання — окрема гілка, оформлена як **Pull Request**.
- Код пишеться в одному файлі; завдання розділяються коментарями.
- Зрозумілі назви змінних, охайне форматування, дотримання **PEP 8**.
- Новий метод чи функція — коментар із поясненням.
- Commit message — інформативний та англійською.
